Git commands:

Make new branch and switch to it:
git checkout -b branchnamehere

View which branch is on:
git branch

How to make a venv:
python -m venv beachcircle_venv

How to activate venv:
beachcircle_venv\Scripts\activate

Get all the stuff in your venv and make it into a requirements.txt (make sure your venv is active first):
pip freeze > requirements.txt

Install based on requirements.txt:
pip install -r requirements.txt

Get the latest changes from main onto your branch:
git fetch origin
git rebase origin/main
